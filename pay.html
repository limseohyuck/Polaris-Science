결제 안내

결제 방식(계좌이체/카드/링크 결제 등)은 안내에 따라 진행됩니다. 결제 오류가 발생하면 실패 페이지 안내를 참고해주세요.

1) 계좌이체

수강 등록 후 안내받은 계좌로 입금해 주세요.

  • 입금자명: 학생명 또는 학부모명 + 학생명 권장
  • 입금 확인 후 수강 확정 안내를 드립니다
  • 현금영수증/증빙 필요 시 미리 요청해 주세요

2) 카드 결제 / 링크 결제

카드 결제(온라인 링크 포함)가 가능한 경우 결제 링크를 별도로 안내드립니다.

  • 결제 링크는 안내된 기간 내에만 유효할 수 있습니다
  • 결제 완료 시 성공 페이지로 이동합니다
  • 오류 발생 시 실패 페이지 안내에 따라 진행해 주세요
테스트 확인용: 성공 페이지 실패 페이지

3) 환불/변경 안내

수업 형태(정규/특강/프로젝트)에 따라 환불 규정이 달라질 수 있습니다.

  • 수업 시작 전/후 환불 기준
  • 교재/재료비 포함 여부
  • 보강 및 일정 변경 기준

4) 사업자 정보 (토스페이먼츠 심사 필수)

아래 정보는 결제 심사 및 고지 의무를 위해 홈페이지에 반드시 표기됩니다.

상호명: 폴라리스 사이언스(Polaris Science Academy)
대표자: 임서혁
사업자등록번호: 여기에 입력 (예: 000-00-00000)
주소: 여기에 입력
고객센터(연락처): 여기에 입력
이메일: 여기에 입력
(해당 시) 통신판매업 신고번호 / 학원(교습소) 등록번호: 여기에 입력
폴라리스 사이언스의 자세한 소식은 네이버블로그를 통해 접하실 수 있습니다.
결제/공지/수업 일정 안내는 블로그에서 확인해주세요.
