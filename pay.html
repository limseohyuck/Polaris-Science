결제 안내

아래 내용은 기본 템플릿입니다. 실제 운영 중인 결제 방식(계좌/카드/링크결제 등)에 맞춰 문구를 더 정확히 다듬어드릴 수 있어요.

블로그에서 공지 확인 홈으로

1) 계좌이체

수강 등록 후 안내받은 계좌로 입금해 주세요.

  • 입금자명: 학생명 또는 학부모명 + 학생명 권장
  • 입금 확인 후 수강 확정 안내를 드립니다
  • 현금영수증/증빙이 필요하면 미리 말씀해 주세요

2) 카드 결제 / 링크 결제

카드 결제(온라인 링크 포함)가 가능한 경우, 결제 링크를 별도로 안내드립니다.

  • 결제 링크는 안내된 기간 내에만 유효할 수 있습니다
  • 결제 완료 시 성공 페이지로 이동합니다
  • 오류 발생 시 ‘실패 페이지’ 안내에 따라 진행해 주세요

3) 결제 성공/실패 테스트 링크

PG/결제 모듈을 붙일 때 아래 페이지로 리다이렉트(redirect)되도록 연결합니다.

성공 페이지 실패 페이지
※ 실제 결제 연동 시: 결제 완료 콜백 URL을 success.html, 실패 콜백 URL을 fail.html로 설정해 사용하면 편합니다.

4) 환불/변경 안내

운영 정책(회차 기준/월 기준/특강 기준)에 따라 문구가 달라집니다.

  • 수업 시작 전/후 환불 기준
  • 교재/재료비 포함 여부
  • 보강 및 일정 변경 기준
원하시면, 서혁님이 이전에 정리하셨던 “학부모 신뢰형 문구(환불 규정 포함)” 톤으로 정식 약관형 문장으로 깔끔하게 맞춰드릴게요.
